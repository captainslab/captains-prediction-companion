# Captain X Article Guide

## Purpose

Convert a completed internal mentions research packet into a public Captain X article/guide.

This skill is only for writing after the research system has produced evidence, TV, picks, live playbook, groups, sneaky NOs, and checklist.

## Do not use when

- rules/source/prices are missing
- the task is still research, not article writing
- the user wants raw analysis rather than public guide format
- final picks have not passed the trade gate

## Format requirements

- plain terminal-friendly text
- all tables inside plain text code boxes
- no sources inside the public article body
- no second-person ownership language
- preserve Captain tone
- include Buy Me a Coffee link placeholder if exact link is not supplied
- end with exact signoff: 🫡🇺🇸💰

## Master public guide structure

```text
[Title block in plain text code box]

Section A — What to expect
Section B — Board
Section C — Best Picks
Section D — Live Playbook
Section E — Groups
Section F — Sneaky NOs
Section G — Quick checklist

Buy Me a Coffee: [link]

🫡🇺🇸💰
```

## Earnings-specific guide structure

```text
[Title block in plain text code box]

Section A — Structure + doc status
Section B — Trade table
Section C — Best picks
Section D — Live playbook
Section E — Groups
Section F — Sneaky NOs
Section G — Quick checklist

Buy Me a Coffee: [link]

🫡🇺🇸💰
```

## Title block pattern

```text
CAPTAIN MENTIONS GUIDE
Event: [event]
Market: [market type]
Date: [date/time]
Focus: exact words, eligible speaker, accepted proof
```

## Section A content

Explain:

- event format
- eligible speaker(s)
- prepared remarks/Q&A/host/moderator/broadcast mechanics
- transcript/source status
- main word clusters
- core caution: exact word, not topic

For earnings, include:

- earnings release / deck / 8-K / transcript status
- whether prepared remarks are known
- how Q&A can force words
- analyst-only warning

## Section B board table

Use code-box table:

```text
| Strike | YES¢ | NO¢ | TV | Edge | Read | Status |
|--------|-----:|----:|---:|-----:|------|--------|
```

Status values:

- TRADE
- WATCH LIVE
- NO TRADE
- FADE SPIKE
- NEEDS PROOF

## Section C best picks

For each best pick:

```text
Pick:
Side:
Entry:
LSP:
Why it works:
How it loses:
Confidence:
```

Keep concise and trade-first.

## Section D live playbook

For each live trade/watch:

```text
Strike:
Buy signal:
De-risk signal:
Kill switch:
```

## Section E groups

Group correlated strikes by theme/exposure:

```text
Group: [name]
Strikes:
Read:
Exposure note:
```

## Section F sneaky NOs

Use for exact-string traps, overbought topic words, analyst-only paths, document-only paths, or paraphrase risk.

```text
Strike:
Why NO is sneaky:
What would kill the NO:
```

## Section G quick checklist

Include a compact checklist:

```text
[ ] Exact word, not topic
[ ] Eligible speaker only
[ ] Transcript/source checked
[ ] Analyst/moderator/guest-only risk checked
[ ] Correlated exposure capped
[ ] Live kill switches known
```

## Language rules

Good:

- "This is a word-proof market, not a topic market."
- "The topic can show up and the strike can still lose."
- "NO TRADE is valid here."
- "Entry must be limit-based."

Avoid:

- "You should buy"
- "Guaranteed"
- "This will hit"
- source dumps inside the public article body
- generic vibes without exact-word mechanics

## Pre-publication gate

Before writing the article, verify internal packet has:

- event snapshot
- contract/rule mechanics
- current prices
- strict word-match grid
- context driver log
- prompt-force map
- paraphrase/dodge map
- market prior board
- evidence TV / MCP / MixMCP board
- trade gate
- live playbook
- groups
- sneaky NOs
- quick checklist

If not, return to research. Do not fabricate article sections.

## Verification checklist

- [ ] guide follows required section order
- [ ] tables are in code boxes
- [ ] no inline sources in public article body
- [ ] every pick has entry, LSP, how it loses, and confidence
- [ ] NO TRADE / WATCH LIVE used where edge fails
- [ ] Buy Me a Coffee line included
- [ ] signoff is exactly 🫡🇺🇸💰
