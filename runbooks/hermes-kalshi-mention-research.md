# Hermes Kalshi Mention Research Runbook

1. Read the board URL and load the Kalshi board.
2. Identify the contract family and the loaded child contracts.
3. Select one child contract to inspect.
4. Find the official source named by the rules.
5. Extract an exact excerpt from the source.
6. Record whether the target phrase is present.
7. Write a board-first record with nested child contract evidence.
8. Save proof in channels/ or state/ if the run changes repo state.

Stop conditions:
- no official source found
- phrase cannot be verified from the official source
- the result would depend on commentary instead of evidence
- the run cannot keep the output board-first
